# ضاد بلوكلي - تطبيق برمجة بصرية

تطبيق ويب للبرمجة البصرية بلغة ضاد العربية باستخدام Google Blockly.

---

## مقدمة

تطبيق ويب يتيح للمستخدمين البرمجة بطريقة بصرية عبر السحب والإلقاء، مع دعم كامل للغة العربية واتجاه RTL.

---

## المميزات

- **برمجة بصرية** - اسحب وأفلت الكتل لبناء البرامج
- **محرر كود** - كتابة الأكواد مباشرة بلغة ضاد
- **دعم العربية** - واجهة كاملة باللغة العربية مع اتجاه RTL
- **تسجيل دخول/تسجيل** - نظام مستخدمين مع حسابات شخصية
- **حفظ تلقائي** - حفظ التقدم تلقائياً في قاعدة البيانات
- **تشغيل الكود** - تنفيذ الأكواد وعرض المخرجات

---

## المتطلبات
<div dir="ltr" align="left">

## التثبيت

```bash
# 1. تشغيل XAMPP
sudo /opt/lampp/lampp start

# 2. نسخ الملفات لمجلد htdocs (إذا لم تكن موجودة)
sudo cp -r . /opt/lampp/htdocs/daad

# 3. تشغيل ملف الإعداد (مرة واحدة)
sudo /opt/lampp/bin/php /opt/lampp/htdocs/daad/setup.php
```
